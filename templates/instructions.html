File Format Instructions

Please follow these instructions carefully to ensure your file is processed correctly.

1. Sheet Name

Your Excel file must contain a sheet named exactly Total stock. The tool will only look for this specific sheet.

2. Header Row

The column headers must be in the second row of the "Total stock" sheet. The first row can be empty or contain a title, but the tool will start reading from the second row as the header.

3. Required Columns

The following columns are required and must be named exactly as shown below. Column names are case-sensitive.

  • Sale Document
  • Item (SD)
  • Material
  • Material Description
  • Plant
  • Storage location
  • Batch
  • Age in Days
  • Unrestricted
  • Value Unrestricted

4. No Bottom Highlights or Totals

Please ensure there are no highlighted total rows or summary calculations at the bottom of the sheet. The tool processes all rows, and these can cause errors.
